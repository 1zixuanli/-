# 实习项目

铁路车票售卖系统前端项目，技术栈为 react-imvc + React。

## 仓库地址

https://github.com/1zixuanli/-.git

## 本地运行

```bash
npm install
npm start
```

浏览器访问：http://localhost:3000

## 功能模块

| 页面 | 路径 |
|------|------|
| 登录 / 注册 | `/login` |
| 首页查询 | `/home` |
| 选城市 | `/city-picker` |
| 车次列表 | `/trainList` |
| 确认订单 | `/order/confirm` |
| 我的订单 | `/orders` |
| 个人中心 | `/profile` |

## 项目结构

```
src/
  login/          登录注册（Controller + Model + View）
  home/           首页
  train-list/     车次列表
  order-confirm/  订单确认
  my-orders/      我的订单
  profile/        个人中心
  api/            接口请求
  data/           Mock 数据
```
